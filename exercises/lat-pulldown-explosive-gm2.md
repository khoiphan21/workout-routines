# Lat Pulldown (Cable) — Explosive

Pull B **technical** work—fast concentric, controlled eccentric; **submaximal** load (not the power triple).

## Pulley setup

| Setting | Value |
|---------|--------|
| **Pulleys** | **~level 7–9** (high) on vertical bars |
| **Attachment** | Lat bar on both cables |
| **Position** | Kneeling (use knee brace on platform if available) |

## How to perform

- **1×5** warm-up, then **3×3** explosive pull; **2–3 s** lowering.
- Keep reps crisp; reduce load if form breaks. Does not replace a muscle-up—trains lat speed.
