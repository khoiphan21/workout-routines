# Prone Incline Cable Y-Raise

Low-incline bench shoulder and lower-trap work using the GM2 cables. Used on Push B before the incline cable press so the bench angle and cable setup stay similar.

## Setup

- Set the bench to a low incline, roughly 30-45 degrees.
- Lie prone on the bench facing the GM2.
- Use low cable handles with light load.

## How to perform

- Keep ribs down and neck neutral against the bench.
- Start with arms slightly in front of the shoulders.
- Raise the handles in a Y path with thumbs up.
- Pause briefly at the top, then lower under control.

## Prescription

- **Power Week:** 3x12, RPE 7-8, giant-set before Incline Bench Press (Cable, GM2) on Push B.
