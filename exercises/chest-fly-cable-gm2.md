# Chest Fly (Cable, Gym Monster 2)

Standing or seated cable fly on the GM2—**H3** on Push A, giant-set with **Overhead Triceps Extension (Cable)** (H4). **H1–H2** use BSS + lateral raise on the floor platform.

## Pulley setup

| Setting | Value |
|---------|--------|
| **Pulleys** | **~level 4–6** on vertical bars (mid) |
| **Attachment** | Smart PowerGrip handles (one per cable) |
| **Bench** | Optional flat Works bench behind you for strict torso |
| **Next in superset** | Raise to high pulley and clip **tricep rope** for OH extension |

## How to perform

- Soft elbow bend; arc hands together at chest height; control the stretch.
- **3×12**, RPE 8–9. Tempo/Volume weeks: optional **bottom partials**.
